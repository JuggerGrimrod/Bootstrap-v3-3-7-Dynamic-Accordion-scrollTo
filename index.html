Some notes about this page...

All accordion drawers coded on this page have a.trigger-link anchors with empty HREF attributes and div.accordion-drawer divs with empty ID attributes:

<a data-toggle="collapse" data-parent="#accordion" href="" class="collapsed trigger-link">CATEGORY NAME X</a>

and

<div id="" class="panel-collapse collapse accordion-drawer">

All a.trigger-link HREF attribute values and div.accordion-drawer ID attribute values are generated dynamically on document.ready by the serializeDrawer() function. serializeDrawer() is called-back on document.ready, and again within the appendAccordion() function when the "APPEND AN ACCORDION DRAWER" button is clicked.
